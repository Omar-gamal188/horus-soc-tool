🛠️ Horus SOC Tool — Installation & Usage Guide
📥 1. Installation
🐧 On Kali Linux
1. Clone the repository
git clone https://github.com/Omar-gamal188/horus-soc-tool.git
cd horus-soc-tool
2. Install pipx (recommended)
sudo apt update
sudo apt install pipx -y
pipx ensurepath
3. Install the tool
pipx install .
🚀 2. Run the Tool
horus
⚠️ Run with root privileges (Required for logs access)
sudo $(which horus)
🧠 Notes

The tool reads logs using:

journalctl

It does NOT rely on:

/var/log/auth.log ❌
Works in real-time (live monitoring)
🧪 3. Testing
